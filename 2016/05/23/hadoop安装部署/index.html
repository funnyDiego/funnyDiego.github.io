hadoop 伪分布式部署
funnyDiego Lv1
 2016-05-23 19:36:05  
  • hadoop 
 2.2k Words  12 Mins  

学习 hadoop 主要参考 hadoop官网，本文主要介绍伪分布式模式的搭建（单节点）及注意事项。

hadoop版本：hadoop-3.2.2

Java版本：jdk 1.8.0_45

1.从官网下载 Hadoop 二进制文件

https://hadoop.apache.org/releases.html 点击 3.2.2 版本 binary => 进入
=> https://dlcdn.apache.org/hadoop/common/hadoop-3.2.2/hadoop-3.2.2.tar.gz 下载 Hadoop 包

2.上传到 ~/tmp 文件夹下（新建）

3.创建用户 hadoop 和常规文件夹

4.将 ~/tmp/hadoop-3.2.2.tar.gz 拷贝到 hadoop 用户下的 software 文件夹

5.解压 tar 包到 ~/app 目录下

hadoop用户（-C：解压到指定目录，解压到当前目录不需要该参数）
[hadoop@bigdata software]$ tar -zxvf hadoop-3.2.2.tar.gz -C ../app/

6.创建软连接

7.配置 JAVA_HOME

8.单机模式测试（无进程，用以debug）

9.伪分布部署

1）配置文件

[hadoop@bigdata hadoop]$ vi etc/hadoop/core-site.xml

[hadoop@bigdata hadoop]$ vi hdfs-site.xml

**2）配置 ssh 无密码登录（重要） **

3）格式化
[hadoop@bigdata hadoop]$ bin/hdfs namenode -format

4）启动 NameNode 和 DataNode

查看进程是否启动成功，jps，但是不保险，最好使用 ps -ef 查看为准

5）访问 Web 界面
hadoop 启动后，访问 hdfs 系统：http://localhost:9870/
根据实际使用的云服务器地址： http://公网IP:9870

6）创建 HDFS 文件夹，执行 Mapreduce Job（形成 hdfs 文件系统，不同于 linux 文件系统）

7）拷贝文件到 hdfs 上，执行 mapreduce job

提交到服务器上执行 mapreduce job
[hadoop@bigdata hadoop]$ bin/hadoop jar share/hadoop/mapreduce/hadoop-mapreduce-examples-3.2.2.jar grep input output 'dfs[a-z.]+'
